MRS Formula

MRS = (0.4×C) + (0.25×R) + (0.25×B) - (0.1×O) + (0.1×L)

Each factor scores 0–5. Note that Operational Friction (O) is subtractive.

Field Mappings

C (Compliance Necessity)

  • FedRAMP Required, CMS ARS Required → 5
  • NIST 800-53 Required, StateRAMP → 4
  • IL4/IL5, Compliance preferred → 3
  • Best practices expected → 2
  • None specified → 1

R (Readiness)

  • Complete → 5
  • In Progress (>75%) → 4
  • In Progress (50-75%) → 3
  • In Progress (<50%) → 2
  • None, Expired → 1

B (Business Value)

  • Very High → 5
  • High → 4
  • Medium → 3
  • Low → 2

O (Operational Friction) - Subtractive

  • High → 5
  • Moderate → 3
  • Low → 1

L (Legal Flexibility)

  • Yes → 5
  • Conditional → 4
  • Negotiable → 3
  • No → 1

Classification Thresholds

  • Migrate Now (MRS ≥ 4.5): Ready for immediate migration
  • Migrate Soon (MRS 3.5–4.4): Near-term action required
  • Migrate Later (MRS 2.5–3.4): Planning phase
  • Hold (MRS < 2.5): Not ready, defer migration

Conditional Award

Records are flagged as Conditional Award Candidates when:

  • Legal Flexibility (L) ≥ 4, AND
  • Readiness (R) ≤ 3

These RFPs can proceed with interim authorization while full ATO is completed.

Data Upload Format

CSV or JSON files must include these fields:

ClientName, Sector, RegulatoryLanguage, ATOStatus, BusinessImpact, OperationalConstraints, LegalFlexibility, Notes
