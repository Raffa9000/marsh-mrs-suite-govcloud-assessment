**Marsh Governance Fabric (MGF)** is a unified decision system for intelligent compliance and audit readiness. It automates the RFP intake, scoring, gap analysis, risk mapping, capacity planning, evidence generation, and orchestration across your GovCloud initiatives.

This repository contains the source code for the MGF suite of 10 lightweight, static web apps built on a canonical data schema and a shared UI framework.
